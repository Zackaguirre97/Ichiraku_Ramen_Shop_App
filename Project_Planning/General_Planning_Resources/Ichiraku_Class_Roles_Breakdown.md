| Class / Enum                                                           | Purpose                                                                                                                                                     |
| ---------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ConsoleController**                                                  | Central controller for the console UI; manages navigation, input handling, and communication between screens and data managers.                             |
| **DataManager**                                                        | Bridges the controller and persistence layer; loads, saves, and updates data models such as items, recipes, customers, and transactions.                    |
| **ConsoleMainScreen**                                                  | Main menu for the console app; routes user actions to ordering, viewing logs, or exiting.                                                                   |
| **ConsoleOrderScreen**                                                 | Displays categories (ramen, drinks, desserts, etc.) for selection and allows adding items to the cart.                                                      |
| **ConsoleBuildYourOwnRamenScreen**                                     | Specialized screen for custom ramen creation; lets users choose broth, noodles, toppings, and extras.                                                       |
| **ConsoleCartScreen**                                                  | Shows current cart contents and totals; allows editing quantities or removing items.                                                                        |
| **ConsoleCheckoutScreen**                                              | Final checkout interface where billing is handled, and the transaction is completed.                                                                        |
| **ConsoleTransactionLogScreen**                                        | Displays all past transactions for review, management, or export.                                                                                           |
| **Item** *(abstract)*                                                  | Parent class for all purchasable products; defines shared fields like `id`, `name`, `price`, and `category`.                                                |
| **Ramen** *(extends Item)*                                             | Represents ramen dishes (preset or custom) with references to ingredient lists or recipe presets.                                                           |
| **Drink** *(extends Item)*                                             | Represents beverage items with attributes like size or type.                                                                                                |
| **Appetizer** *(extends Item)*                                         | Represents side dishes; inherits common item behavior.                                                                                                      |
| **Dessert** *(extends Item)*                                           | Represents desserts available for purchase.                                                                                                                 |
| **Ingredient** *(extends Item)*                                        | Represents a specific ingredient usable in recipes or ramen customization.                                                                                  |
| **Recipe**                                                             | Contains prebuilt ramen or other menu item configurations (not cooking steps).                                                                              |
| **Cart**                                                               | Holds current order contents, calculates totals, and is tied to the checkout process.                                                                       |
| **Transaction** *(immutable)*                                          | Represents one completed sale, containing cart contents, customer info, and billing details. Uses `TransactionStatus` to indicate progress.                 |
| **TransactionLog**                                                     | Stores historical transactions; supports filtering, searching, and data export.                                                                             |
| **BillingInfo**                                                        | Holds payment details such as method, total, and billing name.                                                                                              |
| **Customer**                                                           | Represents the individual making a purchase; contains basic identifying info (name, favorite order, visit count, etc.).                                     |
| **TransactionLogHandler** *(interface)*                                | Defines read/write operations for transaction storage.                                                                                                      |
| **CsvTransactionLogHandler** *(implements TransactionLogHandler)*      | Saves and loads transaction data from CSV files.                                                                                                            |
| **DatabaseTransactionLogHandler** *(implements TransactionLogHandler)* | Handles transaction storage and retrieval from a database.                                                                                                  |
| **ItemFileHandler** *(interface)*                                      | Defines methods for reading and writing `Item` data.                                                                                                        |
| **CsvItemFileHandler** *(implements ItemFileHandler)*                  | Loads and saves item data using CSV.                                                                                                                        |
| **DatabaseItemFileHandler** *(implements ItemFileHandler)*             | Handles `Item` data via a database connection.                                                                                                              |
| **IngredientFileHandler** *(interface)*                                | Defines persistence methods for ingredients.                                                                                                                |
| **CsvIngredientFileHandler** *(implements IngredientFileHandler)*      | Reads/writes ingredient data in CSV format.                                                                                                                 |
| **DatabaseIngredientFileHandler** *(implements IngredientFileHandler)* | Manages ingredient data using a database.                                                                                                                   |
| **RecipeFileHandler** *(interface)*                                    | Defines load/save methods for recipe data.                                                                                                                  |
| **CsvRecipeFileHandler** *(implements RecipeFileHandler)*              | Handles recipe persistence using CSV.                                                                                                                       |
| **DatabaseRecipeFileHandler** *(implements RecipeFileHandler)*         | Manages recipe storage in a database.                                                                                                                       |
| **ReceiptFileHandler**                                                 | Generates receipts for completed transactions, either as text or exportable files.                                                                          |
| **ItemCategory** *(enum)*                                              | Defines categories for menu items (RAMEN, DRINK, DESSERT, etc.).                                                                                            |
| **IngredientCategory** *(enum)*                                        | Defines ingredient types (BROTH, NOODLE, PROTEIN, TOPPING, etc.).                                                                                           |
| **RecipeCategory** *(enum)*                                            | Defines high-level classifications for recipes (PRESET, CUSTOM, LIMITED_EDITION, etc.).                                                                     |
| **TransactionStatus** *(enum)*                                         | Tracks transaction states such as `PENDING`, `COMPLETED`, `CANCELLED`, `FAILED`, or `REFUNDED`.                                                             |
| **CustomerGenerator**                                                  | Utility class that automatically generates randomized `Customer` objects (e.g., ‚ÄúNaruto Uzumaki,‚Äù ‚ÄúSasuke Uchiha‚Äù) for testing or simulated checkout flows. |
